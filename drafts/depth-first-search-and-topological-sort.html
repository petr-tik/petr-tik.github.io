Depth-first search and Topological sort

// under algorithms graphs

There are 2 main graph traversal algorithms - breadth- and depth- first searches. As the name suggests, they explore as far wide or as deep down (respectively )as possible. While each algorithm has its own uses:

  • BFS gives the shortest path (if any exists) between 2 nodes
  • DFS can be used for topological sorting, finding cycles in a graph and other applications.

DFS

The main idea is to explore each child before exploring siblings.
